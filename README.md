# Big Data Lab Practice

This repository contains Big Data lab programs and practice exercises completed as part of my learning.

All the code in this repository was typed and practiced by me for learning purposes. The aim of this repository is to understand basic Big Data concepts and related programming exercises through hands-on implementation.

## Contents

This repository includes practice programs and exercises on:

- Word Count
- Average Length Calculation
- Stop Words
- Bloom Filter
- Matrix Multiplication
- Clustering
- Decision Tree
- Linear Regression
- Logistic Regression
- Support Vector Machine (SVM)

## Purpose

The purpose of this repository is to:

- Practice Big Data lab programs
- Understand basic data processing concepts
- Learn implementation of common algorithms
- Improve hands-on coding practice using Python and R
- Maintain a record of my lab work and learning progress

## Tools and Languages Used

- Python
- R
- Big Data lab concepts
- Machine learning basics

## Note

This repository is mainly for personal learning, lab practice, and revision.
